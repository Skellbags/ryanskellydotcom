What is HTML?

HTML, or hyper-text markup language is a programming language that is the foundation of all modern day websites. It is super easy to learn, its just comprised of Elements! An Element is denoted by < and > wrapped around a certain word that denotes a function for a web client to display. HTML is everywhere! any webpage you go to, you can hit F12 to pull up all of the code behind the page.
